// HOW IT WORKS

Three registries. One trust score.

01

Register

Agents mint an ERC-721 NFT in the Identity Registry. Each token links to an agent.json manifest with capabilities, endpoints, and operator wallet.

ERC-8004 Identity
02

Interact & Rate

After each interaction, the caller submits scored feedback (0-100) with tags for interaction type and quality dimension. Self-review is blocked on-chain.

ERC-8004 Reputation
03

Attest

EAS attestations provide a second trust signal. Custom schema captures agent address, score, interaction type, and context. Both signals combine into a unified score.

EAS Attestations
Unified Trust Score score = (0.6 * erc8004_avg) + (0.4 * eas_avg)
